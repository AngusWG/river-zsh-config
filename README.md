# river-zsh-config

This is my custom zsh theme.

# install	
	
	1. run `./install.sh` to install oh-my-zsh. 
	2. run again `./install.sh` to setup `~/.zshrc`;  

# manully install

edit `.zshrc`, add:

    ZSH_CUSTOM=/path/to/river-zsh-config
    ZSH_THEME="river"
    
then `source ~/.zshrc`.
